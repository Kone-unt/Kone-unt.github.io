Cmd Markdown 高级语法

1.内容目录

在段落中填写@[TOC]来显示全文的内容目录结构。

@[TOC]

2.删除线

使用~~表示删除线。

例如：这个商品原价199，现价9.9

3.注脚

使用[^keyword]表示注脚。

例如：详情请见参考1[^footnote]

4.LateX公式

$表示行内的公式

例如：质能守恒方程的方程式为$E=mc^2​$。

$$表示整行公式：

$$\sum_{i=1}^n a_i=0​$$

$$f(x_1,x_x,\ldots,x_n) = x_1^2 + x_2^2 + \cdots + x_n^2 $$

访问 MathJax 参考更多使用方法。
